Attack Categories

Types of network attacks our system detects

DoS

Denial of Service — Floods the target to make it unavailable.

neptune, smurf, back, teardrop, pod, land
Probe

Surveillance — Scans network to find vulnerabilities.

portsweep, ipsweep, nmap, satan
R2L

Remote to Local — Unauthorized access from a remote machine.

guess_passwd, ftp_write, imap, phf, multihop
U2R

User to Root — Privilege escalation to gain root access.

buffer_overflow, rootkit, loadmodule, perl
